# 1. 投入・整理が必要な基本マスタ

Web.Auto FMSの概念に基づくと、システム稼働のために以下の4つのレイヤーのマスタ情報を整理・登録する必要があります。

1. **車両基本マスタ（静的情報・動的設定）**
* 車両をシステムに認識させ、ファームウェアや車載アプリケーションを紐づけるための基本情報。


2. **地図・環境マスタ（Driving Environment / Map）**
* 自動運転を行うエリア（走行環境）、3D高精度地図（VectorMap）のバージョン管理。


3. **車両固有パラメータ（Calibration / Individual Tuning）**
* 車両ごとに個別にチューニングされた、キャリブレーション用パラメータファイル。


4. **運行・調停制御マスタ（Intersection / Route / Restriction）**
* 信号機のない交差点の通過順序調停（レーン同士の重なり・Right of Way指定）、停留所位置制限、通行禁止・規制区域（時間指定付き）。
