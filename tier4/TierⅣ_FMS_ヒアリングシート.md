## 概要：FMS の主なマスタ構造

FMS では以下の階層でマスタを管理します。

```
プロジェクト
 └─ 走行環境（1プロジェクトに複数設定可）
     ├─ エリアマップ（高精度3Dベクターマップ）
     ├─ 停留所（バス停・乗降地点）
     ├─ ルートパターン（停留所の組み合わせ）
     └─ 車両（走行環境に紐づく自動運転車両）
```
